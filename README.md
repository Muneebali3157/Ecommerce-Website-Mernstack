**🛍️ MK Store - Modern E-Commerce Website**
MK Store is a full-stack e-commerce web application built with the MERN stack (MongoDB, Express.js, React, Node.js) that provides users with a seamless online shopping experience. The platform offers a wide range of products across multiple categories including Home, Fashion, Electronics, Bags, Footwear, Groceries, Beauty, Wellness, and Jewelry.

The application features a responsive design, intuitive user interface, and robust functionality allowing users to browse products, manage their shopping cart, and complete purchases through a secure checkout process. It includes a comprehensive admin panel for store management.
**✨ Features**
**🛒 Shopping Cart**
Add/remove products with quantity selection

Real-time price calculations

Persistent cart using localStorage

Cart preview dropdown in header

Order individual items or all items at once

🔐 User Authentication
Secure user registration and login

JWT token-based authentication

Personalized welcome messages

Session management with login tracking

Role-based access (User/Admin)

**📱 Product Browsing**
Category-based product filtering across 9 categories

Interactive product sliders with Swiper.js

Product detail modal with quick view

High-quality product images with hover effects

Quick view buttons for fast preview

**🔍 Search Functionality**
Real-time search suggestions

Category and product search

Auto-complete functionality

Subcategory search support

**💳 Checkout Process**
Multi-step checkout modal (Shipping → Payment → Review)

Multiple payment options (Card/Bank Transfer)

Shipping information collection

Price breakdown with tax and shipping

Order confirmation with unique order ID

**👑 Admin Dashboard**
Complete store analytics and statistics

Product management (add, edit, delete, track inventory)

Order management with status updates

User management (view, edit, reset passwords)

Low stock alerts and sales tracking

Real-time user activity monitoring

**📱 Usage**
**User Flow**
Browse Products - Navigate through categories on homepage

Search - Use search bar for specific products

View Details - Click any product to see details in modal

Add to Cart - Select quantity and add to cart

Register/Login - Create account or login

Checkout - Proceed to checkout with cart items

Make Payment - Choose payment method and complete purchase

**Admin Flow**
Login - Access admin panel at /admin/login

Dashboard - View store statistics and analytics

Manage Products - Add, edit, or delete products

Manage Orders - View and update order status

Manage Users - Monitor and control user accounts
